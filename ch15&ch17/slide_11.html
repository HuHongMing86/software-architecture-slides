Hybrid Cloud and On-Premise Deployment Is a Unique Strength

Space-Based Architecture supports a powerful hybrid deployment model

  • Cloud Environment: Processing units and virtualized middleware are deployed in the cloud for dynamic, flexible transaction processing.
  • On-Premise Environment: Physical databases and data management remain in a secure, local environment for reporting and data analytics.
  • Enabler: The asynchronous data pump and eventual consistency model make this cloud-based data synchronization highly effective.
This hybrid topology is unique to Space-Based Architecture and provides benefits that other architecture styles cannot easily achieve.
