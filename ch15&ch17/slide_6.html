Processing Units Are Self-Contained Application Modules

  • Application Logic: Contains web-based components and backend business logic. Smaller apps may deploy in a single unit; larger apps split across multiple units.
  • In-Memory Data Grid: A replicated cache implemented via products like Hazelcast, Apache Ignite, or Oracle Coherence.
  • Replication Engine: Ensures data consistency across all active processing units through asynchronous replication (typically under 100ms).
When a processing unit updates its cache, the data grid automatically synchronizes the change to all other processing units containing the same named cache. Member lists track all instances and update dynamically as units start or stop.
